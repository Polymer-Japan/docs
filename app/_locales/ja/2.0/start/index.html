Polycastsを観る

PolycastのChromeデベロッパーリレーションチームからRob Dodsonに参加してください。これは、Polymerの機能の詳細を探る包括的なビデオセットです。 新しいPolymerプロジェクトを一から始めたり、エレメントに機能を追加したり、データバインディングを理解するための特訓を受けたりできます。アクセシブルなコンポーネントを作成し、ターゲットオーディエンス向けにローカライズします。

詳細はこちら

ツールを入手する


npm install -g polymer-cli
mkdir my-app
cd my-app
polymer init

Polymer CLIは、Polymer App Toolboxを使用してPolymerとアプリケーションを構築するためのコマンドラインインターフェースです。ワークフローが合理化され、シンプルなコンポーネントから複雑なWebアプリケーションまで、プロジェクトを簡単に管理できます。
